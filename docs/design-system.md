# Design System

## Brand Direction
- Agriculture-inspired enterprise SaaS.
- Clean operational surfaces with green trust cues and amber action accents.
- Desktop-first with dense but readable spacing for dashboard-heavy work.

## Tokens
- `Primary`: `#166534`
- `Dark`: `#0F172A`
- `Accent`: `#F59E0B`
- `Background`: `#F8FAFC`
- `Border`: `#E2E8F0`
- `Success`: `#16A34A`
- `Danger`: `#DC2626`
- `Font`: `Inter`

## Layout Rules
- Main shell uses a fixed left sidebar and sticky top navbar.
- Content width targets `1440px` while staying fluid below that.
- Cards use rounded corners, subtle glass treatment, and layered shadows.

## UI Patterns
- `Hero headers`: section eyebrow, strong title, supportive description, contextual actions.
- `Metric cards`: bold numeric focus, supporting trend text, soft radial highlight.
- `Data tables`: search, column toggles, pagination, row actions, empty states.
- `Forms`: card-contained sections, visible labels, inline validation, strong submit CTA.
- `Workflow timelines`: vertical progress depiction for advisory and order flows.

## Motion
- Use Framer Motion or subtle transitions only for route loads, modal/dialog reveal, and stateful workflow components.
- Avoid decorative animation on primary work surfaces.
