# Wireframes

## Shared Shell
- `Sidebar`: brand block, grouped role navigation, compact/collapsed state, active highlight.
- `Topbar`: breadcrumbs, global search, notifications dropdown, profile actions.
- `Content`: hero header, metrics, charts, data surfaces, forms, and workflow timelines.

## Authentication
- `Login`: product story panel on the left, auth card on the right, seeded demo credentials, link to forgot password.
- `Forgot Password`: single email field, simulated success state, route back to login.
- `Reset Password`: new password plus confirmation, simulated save state.

## Admin
- `Dashboard`: large hero, KPI grid, revenue chart, query-state donut, low-stock list, audit activity, escalation timeline.
- `Users / Experts / Employees / Vendor Verification`: table-first management screens with search, column toggles, modal forms, row actions.
- `Products / Categories`: catalog governance tables with status badges and create-category modal.
- `Orders / Escalations / Audit Logs`: operational tables with row actions for escalations.
- `Analytics / Reports / Settings`: chart-heavy insight screens, report tables, and an admin settings form.

## Expert
- `Dashboard`: workload metrics, advisory flow timeline, activity feed.
- `Assigned Queries`: searchable case queue.
- `Query Detail`: split screen with case metadata, lifecycle timeline, and recommendation composer.
- `Suggested Products / History / Notifications`: supporting tables and inbox view.

## Employee
- `Dashboard`: KPI grid plus escalation operations context.
- `Task Board`: kanban-like column layout for task stages.
- `Monitoring`: delayed query table with SLA counters.
- `Vendor Support`: support-case table with review/resolve actions.
- `Reports`: employee workload summary table.

## Vendor
- `Dashboard`: order-value chart, KPI cards, low-stock panel.
- `Products`: catalog table with edit links.
- `Add/Edit Product`: two-column form for SKU management.
- `Inventory`: stock monitoring table.
- `Orders / Dispatch`: queue table plus dispatch action cards.
- `Reports`: inventory chart and revenue summary.
